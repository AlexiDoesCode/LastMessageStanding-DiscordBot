# LastMessageStanding


A last message standing discord bot!

## Commands

!start <seconds> - Start a timer, timer resets every time a message is sent to the challenge channel.
  
## Setup

1. Install the requirements (Discord.PY, AsyncIO)
```
pip install discord.py
pip install asyncio
```
2. Open main.py and replace the token variable, and the challenge_channel variable.
